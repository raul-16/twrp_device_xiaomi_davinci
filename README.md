# twrp_device_xiaomi_davinci
TWRP device tree for Xiaomi Mi 9T / Redmi K20

## Features

- Android 13 Support
- ADB
- Decryption of /data (policy V1)
- Screen brightness settings
- Correct screenshot color
- MTP
- Flashing (opengapps, roms, images and so on)
- Backup/Restore
- USB OTG
- Vibration support
- No EROFS support
